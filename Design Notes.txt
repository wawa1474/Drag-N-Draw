# Mouse
### Left:
	\-Place Selected Tile
		|--When Not Clicking On Existing Tile Or Placing Clear Tile
		|--With Color When Not Clear

	\-Move Tile
		|--When Clicking On Existing Tile
		
	\-Copy Clicked On Tile
		|--When Clicking On Existing Tile

### Middle:
	\-Place Colored Tile
		|--When Not Clicking On Existing Tile
		|--No Image, Can't Be Clear

	\-Delete Tile
		|--When Clicking On Existing Tile

### Right:
	\-Recolor tile
		|--When Clicking On Existing Tile

### Scroll:
	\-Switch Which Tile We're Placing

***

# Keyboard:
### WASD:
	\-Move Screen
	
### IJKL:
	\-Reposition All Tiles

### QE:
	\-Change Selected Tile

### Space-Shift:
	\-Change Tile Row

### X:
	\-Cut Group Selection

### C:
	\-Copy Group Selection

### V:
	\-Paste Group Selection

### F:
	\-Toggle Tile Clear

### R:
	\-Console Log The Properties Of The Tile The Mouse Is Over
	
### P:
	\-Set tile group XY1 and XY2
