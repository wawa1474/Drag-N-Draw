# Mouse
### Left:
	\-Place Selected Tile
		|--When Not Clicking On Existing Tile
		|--With Color When Not Clear

	\-Move Tile
		|--When Clicking On Existing Tile
		
	\-Copy Clicked On Tile
		|--When Clicking On Existing Tile

### Middle:
	\-Place Colored Tile
		|--When Not Clicking On Existing Tile
		|--No Image, Can't Be Clear

	\-Delete Tile
		|--When Clicking On Existing Tile
### Right:
	\-Recolor tile
		|--When Clicking On Existing Tile





***

# Keyboard:
### WASD:
	\-Move Screen
	
### IJKL:
	\-Reposition All Tiles

### QE:
	\-Change Selected Tile

### Z-X/Space-Shift:
	\-Change Tile Row

### C:
	\-Toggle Tile Clear
	
### P:
	\-Set tile group XY1 and XY2
